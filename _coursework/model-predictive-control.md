Graduate-level course on Model Predictive Control (MPC) theory and implementation for constrained dynamical systems, with applications to robotics and mechatronic platforms.

## Topics Covered

* State-space modeling and system identification
* Optimal control and receding horizon formulation
* Constrained optimization (linear and nonlinear MPC)
* Stability, feasibility, and robustness analysis
* Real-time implementation strategies
* Applications to robotic and autonomous systems

## Course Project

Implemented MPC and LQR algorithms using CasADI for a two-wheeled balancing robot, handling coupled yaw dynamics, incline navigation, and actuator torque constraints. Validated via MuJoCo simulation.

## Skills Developed

* Constrained optimization and solver frameworks (CasADI)
* Real-time control system design
* Dynamic modeling and simulation (MuJoCo)
* State estimation and feedback control
